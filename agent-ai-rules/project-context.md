# PITLANE F1 2026 — FINAL PROJECT CONTEXT

# Project Overview

Pitlane is a cinematic multi-page Formula 1 web application built around the 2026 FIA Formula 1 World Championship season.

The application is designed to feel like:
- a premium Formula 1 digital platform
- a futuristic race command center
- a modern motorsport companion app
- a highly animated sports experience

The project combines:
- beginner-friendly React architecture
- advanced frontend presentation
- cinematic motion design
- immersive user experience
- responsive premium UI

---

# Core Philosophy

## IMPORTANT

The internal React logic must remain:
- simple
- educational
- beginner-readable
- easy to maintain

However, the frontend experience must feel:
- advanced
- polished
- cinematic
- immersive
- professional

The goal is:
> simple logic + premium presentation

---

# Developer Skill Level

The developer is beginner/intermediate in React.

Because of this:

## Prioritize

- readability
- maintainability
- understandable state
- simple component logic
- clean props flow
- reusable UI

## Avoid

- over-engineering
- complex abstractions
- unnecessary custom hooks
- advanced architecture patterns
- deeply nested systems

---

# Preferred React Concepts

The project should mainly rely on:

- useState
- previous state callback
- spread operator
- map()
- filter()
- find()
- some()
- props
- event handling
- useEffect
- localStorage
- JSON.stringify()
- JSON.parse()
- dynamic routes
- useParams()
- key prop

The project should remain educational and easy to explain during presentation.
